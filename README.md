# Sistema de Gestión de Citas – Frontend

Aplicación web desarrollada en React para la gestión de citas, clientes y servicios de un centro terapéutico.  
Este proyecto corresponde a un trabajo freelance realizado de manera integral, desde el análisis de requisitos hasta el despliegue.

## Tecnologías
- React
- JavaScript
- Tailwind CSS
- HTML5 / CSS3
- Git & GitHub

## Funcionalidades
- Gestión de clientes y servicios
- Agendamiento y visualización de citas
- Generación y exportación de documentos PDF generados con base a necesidades
- Visualización de métricas semanales y mensuales
- Interfaces dinámicas y responsivas
- Consumo de APIs REST

## Entorno de prueba
La demo utiliza una base de datos independiente en un entorno de pruebas, separada del entorno productivo del cliente.
